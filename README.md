# Assignment-0
1. Clone this repo locally to your computer.
2. Read the friendly manual (how-to-tex.pdf). Note that the .tex file used to create this document is included for you to reference!
3. Read the instructions (A0-Instructions.pdf).
4. Edit the template file (A0.tex) with what the instructions are asking for.
5. Compile the tex file to create a pdf. **This pdf, not the .tex file itself, is your submission!**
6. Commit those edits and push the repo back up to GitHub to submit your work!
